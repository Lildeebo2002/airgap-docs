If you hold Coreum in AirGap, you can delegate them to earn rewards.

| Action               | Value               |
| -------------------- | ------------------- |
| Locked while staking | Yes                 |
| Undelegating period  | 7 days              |
| Rewards after        | Immediately         |
| Slashing Risk        | Yes                 |
| Payout               | Needs to be claimed |
