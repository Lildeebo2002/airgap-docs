If you hold ICP in AirGap, you can delegate them to earn rewards.

| Action               | Value                             |
| -------------------- | --------------------------------- |
| Locked while staking | Yes                               |
| Undelegating period  | Dynamically set by the user       |
| Rewards after        | Immediately if followee has voted |
| Slashing Risk        | No                                |
| Payout               | Added to the stake automatically  |
