## Introduction to Internet Computer Protocol

ICP is an open-source decentralized network that aims to revolutionize the internet by offering a new way of building and running applications. This next-generation computing infrastructure provides scalability, decentralization, security, and high performance.

ICP uses the unique "Threshold Relay" consensus mechanism for fast, secure consensus. The network also features the Internet Computer virtual machine, providing a secure and scalable platform for hosting decentralized applications. [learn more](https://internetcomputer.org/)

Founded by [Dfinity](https://dfinity.org/), a non-profit organization dedicated to advancing decentralized technology. Their goal is to create a new internet that is more secure, decentralized, scalable, and innovative.

ICP is the native currency of Internet Computer Protocol.
