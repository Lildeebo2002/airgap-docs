## Multiple seedphrase support

Most cold wallets support one seed phrase. AirGap starts where other cold wallets stop. With AirGap, you can easily manage and organize multiple wallets for family, savings, defi, personal, or any other purpose, all within one app that will be offline.

Plus, managing multiple wallets has never been easier! AirGap's intuitive UI lets you switch between wallets seamlessly.
